# What is this project?

This project was a take home coding challenge: **Build a dashboard to display messages from a server.** 

The requirements were: 
1. Display the messages
2. Filter messages by `level`
3. Start/stop messages from the server
4. Delete all messages
5. Delete individual messages
6. Edit messages
7. Use pretty colors
